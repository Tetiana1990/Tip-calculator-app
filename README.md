                           # Tip Calculator
  This is a responsive Tip Calculator built with HTML, CSS, and JavaScript.
  It allows users to calculate tips per person or total amount depending on the 
  bill, selected tip percentage, and number of people.

                            #Features
  .Mobile-first design with responsive adjustments for desktop.
  .Tip selection buttons(5%, 10%, 15%, ets.) with active state highlighting.
  .Custom tip input for entering a specific percentage.
  .Dynamic calculation:
      -Shows tip per person when number of people is entered.
      -Defaults to 1 person if people input is empty.
   .Error handling:
       -Displays "Can't be zero" when number of people = 0.
       -Highlights input wrapper with red border.
  .Reset button:
       -Clears bill,people, custom tip, and selected buttons.
       -Removes error messages and red border.
  .Interactive states for all buttons and inputs(hover,focus).

                           #Technologies Used
   .HTML5- Semantic markup for inputs, labels, buttons
   .CSS3- Mobile-first responsive styling, flexbox and pseudo-classes for focus/hover states
   .JavaScript(ES6)- DOM manipulation, event listeners, dynamic calculation, error handling

                            #Usage
    1.Enter the bill amount.
    2.Select a tip percentage button or enter a custom tip.
    3.Enter the number of people(optinal, defaults to 1).
    4.View the tip per person and total per person.
    5.Press RESET to clear all fields and start over.
